About SatLoom

A free, open-source communication platform using WebRTC technology.

Version: 2.0.0

Developed by: SatLoom Developers

Features:

  • High-quality audio and video calls
  • Secure text chat
  • End-to-end encryption
  • Cross-platform compatibility
  • Open source and free to use

Technology Stack:

  • WebRTC for real-time communication
  • Firebase for signaling and room management
  • STUN/TURN servers for NAT traversal
